Aprenda como inicializar, construir e implantar edge applications na plataforma de Edge da Azion através da linha de comando.

Com a **Azion CLI**, você pode gerenciar e interagir com suas applications na plataforma.

A Azion disponibiliza um conjunto de opções para interagir com os produtos da Azion, tais como:

- [Azion Terraform Provider](/pt-br/documentation/products/terraform-provider/)
- [Azion API](https://api.azion.com/)
- [Azion SDK (Go)](/pt-br/documentation/devtools/sdk/go/)

A Azion CLI é desenvolvida em Go e interage com os produtos da Azion através do [Azion SDK (Go)](/pt-br/documentation/devtools/sdk/go/).

## Instalando a Azion CLI

### Download

Se você optar pelos gerenciador de pacotes **RPM**, **Dpkg** ou **apk**, vá para a [página de lançamentos](https://github.com/aziontech/azion-cli/releases) e faça o download do pacote desejado.

:::note 
Se você escolher usar o Homebrew, a etapa anterior não será necessária.
:::

Escolha uma das seguintes opções:

## Atualizando a Azion CLI

Toda vez que uma nova versão é lançada e qualquer comando da Azion CLI é utilizado, um prompt informará o usuário sobre a nova versão e perguntará se ele deseja atualizá-la ou não.

## Opções globais

### version 

```sh 
  azion --version
```

A opção `-v` ou `--version` informa qual versão da Azion você tem instalada.

### debug 

```sh 
  azion --debug
```

A opção `-d` ou `--debug` exibe logs no nível de depuração.

### config 

```sh 
  azion --config
```

A opção `-c` ou `--config` define a pasta de configuração da Azion apenas para o comando atual, sem alterar as configurações persistentes.

### log-level 

```sh 
  azion --log-level info
```

A opção `-l` ou `--log-level` informa o nível de log a ser exibido. As opções são:

- `info` (padrão)
- `error`
- `debug`

### silent 

```sh 
  azion --silent
```

A opção `-s` ou `--silent` silencia completamente os logs. Normalmente, é usada para fins de automação.

### token 

```sh 
  azion --token seu-token-pessoal
```

A opção `-t` ou `--token` salva um token pessoal fornecido localmente para autorizar comandos do CLI.

### yes 

```sh 
  azion --yes
```

A opção `-y` ou `--yes` responde automaticamente com sim a todas as interações sim/não.

#### help

A opção `h` ou `--help` exibe mais informações sobre o comando `azion`.
