Veja o comando que permite a inicialização de um servidor de desenvolvimento local atrvés da Azion CLI.

---

## dev

O comando `dev` adapta o ambiente para a build de sua edge application e a roda o projeto localmente.

### Uso

```sh
azion dev
```

### Saída 

Depois de inicializar o servidor de desenvolvimento local, a aplicação passa pelo processo de `build`.

```sh 
Building your Edge Application. This process may take a few minutes
Running build step command:
...
```

Em seguida, quando a build estiver completa, o acesso à aplicação é apresentado:

```sh
[Vulcan] [Servidor] › ✔  sucesso   Função em execução na porta http://localhost:3000 
```

### Flags opcionais

#### help

A opção `--help` exibe mais informações sobre o comando `dev`.
